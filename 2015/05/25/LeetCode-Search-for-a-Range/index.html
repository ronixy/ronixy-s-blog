Search for a Range

一个有序数组，在log(n)的时间找出指定的某个数第一次出现与最后一次出现的位置。

用了递归二分查找，分别找出要求的两个下标，因为求mid时左右相加再除以2向下取整这个特性，所以找最后一次出现下标和找第一次出现时下标的判定条件略有不同，代码中有标注，要注意。
感觉写的有些麻烦，或许可以写到一个函数里，回头再想想吧。。
