[LeetCode] Reverse Integer/String to Integer (atoi)

Reverse Integer

简单题，注意判断溢出，我这里用了long long 表示结果，再将其赋给一个int，然后判断两个数是否相等，若相等则没有溢出，否则就溢出。

leetcode给的solution给了一种判断溢出的方法：在一个数乘以10之前，先看它是否大于214748364（int最大是2147483647，补码表示）或者小于 –214748364（int最小-2147483648，补码表示）。感觉这个方法更好用。

String to Integer (atoi)

实现将字符串转换为整数的atoi函数，需要注意几点：

  • 从第一个不是空格的字符开始；
  • 若第一个不是空格的字符不是数字或者正负号，则返回0，比如” a123”要返回0（这里坑了我好久）；
  • 若正负号后面紧跟的不是数字，也返回0，比如”+-2”要返回0；
  • 一直到第一个不是数字的字符截止，比如” +123abc321”返回123；
  • 判断溢出，溢出则返回离它最近的数（INT_MAX或INT_MIN）；
