[LeetCode] ZigZag Conversion/Palindrome Number

ZigZag Conversion

给一个字符串和一个n，把字符串变成n行的之字形状，用了笨方法，搞了n个string，挨个遍历字符放入对应的string中，然后从按顺序输出就成了。

其实也有比较简单的数学规律，不过也还是要遍历一遍，也要开n个string，还不如我这笨方法简单易懂^_^

Palindrome Number

这题要求不能用多余的空间，就是不让人用数组存每位数字然后再头尾比较，空间复杂度为O(1)。方法还是比较简单易懂的。
