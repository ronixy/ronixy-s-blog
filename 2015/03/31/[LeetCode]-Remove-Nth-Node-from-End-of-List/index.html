Remove Nth Node From End of List

设置两个指针p和pre，都从链表头往后遍历，当p遍历到第n个结点时，pre开始遍历，这样当p到链表尾部时，pre指向的恰好就是倒数第n个结点。

要删除倒数第n个结点，其实是要得到倒数第n+1个结点的指针的，所以实际上是p遍历到第n+1个结点时pre才开始遍历的，这样就可以直接删除倒数第n个结点。
还要注意要求删除第一个结点的情况，设置了一个计数器cnt，随p增加，最后当p到尾部时cnt也就是链表的长度，如果n == cnt说明要删除的是第一个结点，这时直接返回head->next就行了。
复杂度O(n)。
